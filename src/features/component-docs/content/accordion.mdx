<h2 id="usage">Usage</h2>

`AccordionRoot` 안에 `AccordionItem`, `AccordionTitle`, `AccordionContent`를 조합합니다. 열린 항목은 `values`로 초기화합니다.

<h2 id="examples">Examples</h2>

<h3 id="single">Single</h3>

`multiple={false}`를 사용하면 한 번에 하나의 항목만 펼칩니다.

<ComponentPreview sourceId="accordion-examples">
  <AccordionSingleExample />
</ComponentPreview>

<h3 id="rounded">Rounded</h3>

`rounded`를 지정하면 첫 항목과 마지막 항목의 바깥 모서리가 둥글게 표시됩니다. 위 기본 예제에서 확인할 수 있습니다.

<h2 id="props">Props</h2>

- `values: string[]` — 처음 열 항목
- `multiple?: boolean` — 여러 항목 동시 열기, 기본값 `true`
- `rounded?: boolean` — 바깥 모서리 처리
- `onChange?: (values: string[]) => void` — 열린 항목 변경 콜백

<h2 id="known-limitations">Known limitations</h2>

현재 `AccordionTitle`은 `div` 클릭으로 동작해 키보드와 ARIA 상태를 제공하지 않습니다. 현행 UI 보존 범위에서는 수정하지 않았으며, 접근성 개선 작업에서 button 기반 trigger로 교체해야 합니다.
