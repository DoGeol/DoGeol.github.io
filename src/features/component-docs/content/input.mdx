<h2 id="props">Props</h2>

- `size?: 'small' | 'medium' | 'large'` — 입력 높이, 기본값 `medium`
- `onEnter?: (event, value) => void` — Enter 입력 시 호출
- `wrapperClass?: string` — 바깥 wrapper class
- 그 밖의 표준 `input` 속성

<h2 id="known-limitations">Known limitations</h2>

현재 비밀번호 표시 버튼은 접근 가능한 이름과 명시적 `type="button"`이 없습니다. 현행 UI 보존 범위에서는 수정하지 않았으며, 접근성 개선 작업에서 보완해야 합니다.
